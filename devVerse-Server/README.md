## 📌 Overview

This folder contains the **server-side code** of **DevVerse**, a modern **developer forum website**.  

It provides the **backend API** for posts, blogs, DevStudio, WebStudio, messaging, notifications, user management, payments, and leaderboard functionalities.  

**Keywords:** Node.js, Express, Backend, API, MongoDB, Developer community, Authentication, Realtime messaging, DevStudio, WebStudio.

<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif" width="100%">

## 📖 Features

- 👤 User authentication and role management  
- 📝 CRUD operations for posts, blogs, and comments  
- 🏗️ DevStudio & WebStudio code execution APIs  
- 💬 Real-time messaging and video call support  
- 🏆 Leaderboard & engagement tracking  
- 💰 Payment and premium membership handling  
- 🛑 Report handling and moderation  
- 🔔 Notifications system

## ⚙️ Installation
1. Clone the repository (main repo):
```bash
git clone https://github.com/Gaurab1809/DevVerse-A-Developer-Community-Platform-Website.git
```
2. Navigate to the server folder:
```bash
cd devVerse-Server
```
3. Install dependencies:
```bash
npm install or npm i
```
4. Start the server:
```bash
nodemon index.js
```
The server will run at http://localhost:5000
 (default). Make sure MongoDB is running and properly configured.

Note: The frontend (client) needs this server API to function correctly.

<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif" width="100%">

## 📝 Notes
The node_modules/ folder is not included in GitHub and will be installed via npm install.
Keep MongoDB and server running to support all frontend features like posts, blogs, messaging, DevStudio, WebStudio, leaderboard, and notifications.
